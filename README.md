## Getting Started

### Installing

1. Clone the repository:

```bash
git clone https://github.com/Theanordhoy/portfolio.git
```

2. Navigate to the project folder

```bash
cs portfolio
```

3. Open the project in code editor

```bash
code .
```

4. Start the project using Live Server
